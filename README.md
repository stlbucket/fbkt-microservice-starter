# fbkt-microservice-starter
create an fbkt microservice

requirements:
```
-node ^6.9
```

usage:
```
git clone https://github.com/stlbucket/fbkt-microservice-starter myAppName
cd myAppName
npm install
npm run buildDb
npm run test
npm run dev
```

**/config/dev/index.js** contains db connection setup - alter accordingly
